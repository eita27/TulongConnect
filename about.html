About TulongConnect

NGO donation platform prototype overview

TulongConnect is a mobile-first donation platform that connects donors with NGOs through structured donation planning, inventory transparency, and role-based workflows.

Core features in this prototype:

  • Role-based sign in: Donor or NGO path
  • Protected routes: only Login and Sign Up are public
  • Donation planner with category and quantity inputs
  • Donor donation history (recipient + items + quantity)
  • NGO inventory monitoring by category stock level
  • NGO donor request tracking with status visibility

Open index.html to run the interactive prototype.
